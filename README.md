# DEW International Schools Website

Official website for DEW International Schools.

## Overview

A production-ready school website designed to provide information about the institution, facilitate enquiries, and improve communication between the school and prospective parents.

## Features

- Responsive design
- School information pages
- Admissions information
- Contact forms
- Google Sheets integration for enquiry management
- WhatsApp integration for direct communication
- Mobile-friendly interface
- Custom domain deployment

## Technologies

- HTML5
- CSS3
- JavaScript
- Google Sheets
- Netlify

## Key Integrations

### Enquiry Management
Website form submissions are automatically stored and managed through Google Sheets.

### WhatsApp Communication
Parents can initiate conversations directly with the school through integrated WhatsApp contact links.
